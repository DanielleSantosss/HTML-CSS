Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag <b> não semântica.

Nesta frase, temos um termo em negrito usando a tag <strong> com semântica.

Itálico/Ênfase

Nesta frase, temos um termo em ítalico usando a tag <i> não semântica.

Nesta frase temos um termo em ítalico usando a tag <em> com semântica.

Texto marcado

Podemos criar um texto marcado com a tag <mark>.

E no outro parágrafo, temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande (essa tag <big> não é mais usada) e um texto pequeno usando a tag <small>.

Texto deletado

Podemos marcar como um texto excluído para marcar que ele deve ser lido, mas não considerado, usando a tag <del>.

Texto inserido

Podemos marcar um texto inserido para dar ênfase e indicar que ele foi adicionado depois usando a tag <ins> com valor semântico.

Existe também o sublinhado com a tag <u> não semântico.

Texto sobrescrito

Para inserir coisas do tipo x20+3 se usa a tag <sup>.

Texto subescrito

Para inserir coisas do tipo H2O se usa a tag <sub>.
